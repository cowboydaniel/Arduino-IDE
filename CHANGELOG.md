# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Initial project structure
- Code editor with Arduino C/C++ syntax highlighting
- Line numbers and current line highlighting
- Auto-indentation for C/C++ code
- Serial monitor with multi-device support
- COM port auto-detection
- Configurable baud rates (300-250000)
- Three built-in themes (Dark, Light, High Contrast)
- Project explorer panel
- Board information panel
- Console panel for build output
- Variable watch panel for debugging
- Tabbed editor for multiple files
- Dockable panels for flexible layout
- Window state persistence
- Menu system with keyboard shortcuts
- Toolbar with quick actions
- Status bar
- Example Arduino sketch templates:
  - Basic sketch
  - Sensor reading
  - Button control with debouncing

### Documentation
- Comprehensive README with features and installation
- Architecture documentation
- Contributing guidelines
